# Instagram-Clone


 
 This Instagram clone allow user to create their account , login their account. The main functionality
 of this application is user can put their statuses, edit their profile as well as their bio and many more
 features. I have used the dependencies like express , passport, cookie-parser , express-session etc.
 I have used the technology in the backend like Node Js, Express Js and MongoDB for database, Axios
 to enhance the user experience and create dynamic , responsive interactions within this project. I
 have used asynchronous concept to effortlessly fetch and update data without reloading the entire page.
